=== describeNumber ===
-5 -> отрицательное нечётное
-2 -> отрицательное чётное
0 -> ноль
1 -> положительное нечётное
4 -> положительное чётное
7 -> положительное нечётное

=== describeString ===
"" -> пустая строка
"hello" -> длина: 5, нижний регистр
"HELLO" -> длина: 5, ВЕРХНИЙ РЕГИСТР
"HeLLo" -> длина: 5, смешанный регистр
"java123" -> длина: 7, содержит цифры, нижний регистр
"Hello World" -> длина: 11, смешанный регистр

=== getObjectType ===
Hello -> Строка
42 -> Целое число
3.14 -> Дробное число
true -> Логический тип
null -> null
